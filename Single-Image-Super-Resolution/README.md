# Single Image Super Resolution using CNN and GAN

## Description
This project implements Single Image Super Resolution (SISR)
using SRCNN and SRGAN to reconstruct high-resolution images
from low-resolution inputs.

## Tech Stack
- Python
- PyTorch
- OpenCV
- NumPy
- Matplotlib

## Project Structure
- notebook/: main Jupyter notebook
- src/: model and dataset code
- sample_images/: example inputs
- results/: output comparisons

## How to Run
1. Install dependencies:
   pip install -r requirements.txt
2. Open notebook/sisr_project.ipynb
3. Run all cells

## Results
SRGAN produces visually sharper images compared to SRCNN.
